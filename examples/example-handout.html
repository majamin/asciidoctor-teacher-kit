Let’s test some Go

Every test is written in a file whose name ends with _test.go
func Test_walks(t *testing.T) { (1)
  result := walks(4,4)
  if result != 20 {
    t.Error("incorrect result: expected 20, got", result) (2)
  }
}
1 Requires the testing package
2 Error call

The relationship

The number of staircase walks on a grid with \$m\$ horizontal lines and \$n\$ vertical lines is given by

\[ \left(\frac{m+n}{n}\right) = \frac{(m+n)!}{m!n!}\]
